# 5GModel

Regression test for 5GModel includes

1) Waveform compliance test against MATLAB 5GToolBox
2) TV generation for cuPHY and cuBB verification
3) Receiver performance simluation based on 3GPP spec

```matlab
     cd <5GModel root>/nr_matlab/
     startup
     runRegression
```

The test vectors will be located in <5GModel_root>/nr_matlab/GPU_test_input
